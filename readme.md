## _CryptoTool _使用
```cpp
git clone https://github.com/wcwq15232/HelloCV
cd HelloCV/Crypto
mkdir build && cd build
cmake ..
cmake build .
make
cmake --install . --prefix ../install_testpath
cd ../install_testpath/bin
./CryptoTool
```
